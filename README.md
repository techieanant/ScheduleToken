
# A Simple ERC-721 ScheduleToken Example

What is it?

A Schedule Token can be minted by the owner of the contract which can be redeemed by anyone for an appointment. 

### Setup the project

Make sure that you have node, npm and truffle installed then

    npm install
    

Install [Ganache](http://truffleframework.com/ganache/) and make sure it is running on 8545

Compile and Migrate

    truffle compile
    truffle migrate
